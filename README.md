# Typing Speed Checker ⌨️

A simple Typing Speed Checker built using **HTML, CSS, and JavaScript**.  
It measures typing speed (WPM), accuracy, and errors in real time.

## Features
- Random typing text
- 60-second timer
- Live error tracking
- Words Per Minute (WPM) calculation
- Accuracy percentage
- Restart functionality

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## How It Works
- Timer starts on the first keystroke
- User input is compared character by character
- WPM is calculated using the standard 5-characters = 1 word rule
- Results are displayed when the timer ends
